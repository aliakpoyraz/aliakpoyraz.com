## 1. Kod Blokları ve Syntax Highlighting

Bir yazılımcı bloğunun en önemli kısmı kod bloklarıdır. Aşağıda farklı dillerde örnekler var.

**JavaScript / TypeScript:**

```typescript
interface User {
  id: number;
  name: string;
  role: "admin" | "guest";
}

function getUser(id: number): User {
  // Veritabanından kullanıcıyı çekme simülasyonu
  return {
    id: id,
    name: "Ali Akpoyraz",
    role: "admin",
  };
}

console.log(getUser(1));
```

**Python:**

```python
def veri_analizi(data):
    """
    Veri setini analiz eder ve sonuçları döndürür.
    """
    if not data:
        return None
    
    sonuc = [x * 2 for x in data]
    return sonuc

print(veri_analizi([1, 2, 3, 4]))
```

**CSS:**

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  background-color: #0c0c0e; /* Koyu tema */
}
```
